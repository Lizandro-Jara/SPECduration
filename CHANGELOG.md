# Changelog

## [0.1.0] – 2026-02-05
### Added
- Initial public release of **drs-duration**
- Command-line interface (CLI) for seismic duration estimation
- High-resolution Displacement Response Spectrum (DRS) computation using Newmark-β
- Automatic detection of the predominant harmonic window
- α-scan procedure for optimal derivative threshold selection
- Nonlinear sinusoidal fitting with optional constant offset
- Generation of reproducible CSV tables and publication-ready figures
